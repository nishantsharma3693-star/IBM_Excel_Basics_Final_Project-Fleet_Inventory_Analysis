# IBM_Excel_Basics_Final_Project-Fleet_Inventory_Analysis

# Objective

The main objective of this project was to clean and organize messy fleet inventory data and convert it into a structured format for better analysis. The project also aimed to create Pivot Tables to analyze equipment distribution across departments and vehicle categories.

# Steps Taken to Clean & Remove the Inconsistencies from Data

1. Removed duplicate and unnecessary records from the dataset.
2. Corrected inconsistent formatting and aligned the data properly.
3. Checked and handled missing or blank values.
4. Standardized column names and category labels for better readability.
5. Organized the dataset into a tabular format suitable for analysis.
6. Verified data accuracy before creating reports.
7. Created Pivot Tables to summarize department-wise and equipment-wise information.

# Results Derived

1. Obtained a clean and well-structured dataset for analysis.
2. Generated Pivot Tables to analyze fleet equipment inventory efficiently.
3. Identified the number of vehicles and equipment under different departments.
4. Categorized equipment types such as SUVs, Sedans, Trucks, and others.
5. Improved data readability and reporting accuracy.
6. Made the dataset easier for decision-making and inventory tracking.

# Key Learnings

> Learned practical data cleaning techniques in Excel.
> Improved understanding of handling messy and inconsistent datasets.
> Gained hands-on experience in using Pivot Tables for data analysis.
> Learned how to summarize large datasets into meaningful insights.
> Enhanced skills in data organization, formatting, and reporting.
> Understood the importance of clean data for accurate analysis and visualization.
